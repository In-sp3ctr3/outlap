# License Recommendation

Recommended license for v1: Apache-2.0.

Why:

- permissive and contributor-friendly;
- includes patent grant;
- familiar to open-source communities;
- works well for plugin and integration ecosystems.

Before first public release, maintainers may choose AGPL-3.0-or-later instead if preventing closed hosted forks is more important than permissive adoption. Decide before accepting external contributions.

Implementation agent should add the full Apache-2.0 `LICENSE` file during repository creation.
