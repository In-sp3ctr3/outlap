# ADR-0008 — Use MIT License for v1

## Status

Accepted.

## Context

The project wants open-source adoption, integrations, and community contributions. The maintainer selected MIT before public release.

## Decision

Use MIT for v1.

## Consequences

Positive:

- permissive;
- familiar;
- easier for contributors and downstream users.

Negative:

- does not include an Apache-style explicit patent grant;
- hosted closed-source forks are possible.

Mitigation:

- build strong community and governance.
