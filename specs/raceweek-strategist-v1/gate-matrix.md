## Current Implemented Scope

The current implementation includes repository hygiene, a local demo product, and the Milestone 1 DuckDB contracts/database slice. Deterministic rules, projections, optimizer, manual team/market/league/score imports, optional read-only fantasy access, mocked OpenF1 session-context fetching, mocked Jolpica season-context fetching, metadata-only RSS/news ingestion, a FastF1 cache adapter, data health degradation, fake provider fallback, league analysis, and the multi-page UI are present for demo workflows.

The full v1 spec is not yet complete. Full provider SDK calls, full projection feature/backtest engine, OR-Tools optimizer path, and release-grade QA/docs remain open work.
